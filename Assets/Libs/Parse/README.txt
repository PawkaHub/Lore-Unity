The official Parse.com Unity SDK. Add a cloud backend to your Unity games with this simple-to-use SDK.



Add a cloud backend to your Unity games in minutes.

Features:
* Parse Data - Store your app's data in the cloud. No servers necessary.
* Parse Social - Make your app social. Instantly.
* Cloud Code - Run custom app code in the Parse Cloud. Say goodbye to servers.
* Supports iOS, Android, Desktop, and Web Player games.
